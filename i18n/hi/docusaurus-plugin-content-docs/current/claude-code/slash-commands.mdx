## ये कैसे काम करते हैं

एक कमांड बस एक Markdown फ़ाइल है। इसे अपनी commands डायरेक्टरी में डालें और इसका फ़ाइलनेम ही कमांड बन जाता है। फ़ाइल का बॉडी वह प्रॉम्प्ट है जिसे आप इसे इनवोक करने पर Claude चलाता है।

## एक न्यूनतम उदाहरण

यहाँ एक पूरी कमांड फ़ाइल है। फ्रंटमैटर में `description` होता है; इसके नीचे जो कुछ भी है वह प्रॉम्प्ट है।

<PromptCard title="commit.md — एक Conventional Commit ड्राफ़्ट करें">{`---
description: Draft a Conventional Commit message from the staged diff
---

Look at the staged changes with \`git diff --cached\` and write a single
Conventional Commit message (feat/fix/docs/refactor/chore). Output only the
message, no preamble.`}</PromptCard>

इसे `commit.md` के रूप में सेव करें और `/commit` इसे हर बार कर देता है। (फ़ाइल जो भी हो उसी नाम से रखें — वही बॉडी अगर `changelog.md` के रूप में सेव की जाए तो वह `/changelog` होगी।)

## उपयोगी बिल्डिंग ब्लॉक्स

चार सामग्रियाँ एक स्थिर प्रॉम्प्ट को एक लचीली कमांड में बदल देती हैं:

| बिल्डिंग ब्लॉक | यह क्या करता है |
| --- | --- |
| **`$ARGUMENTS`** | उपयोगकर्ता ने कमांड के बाद जो टाइप किया उसे इंजेक्ट करता है। `/review src/auth.ts` → वह पाथ आपके प्रॉम्प्ट में आ जाता है। |
| **bash चलाएँ और आउटपुट एम्बेड करें** | कमांड्स शेल आउटपुट (जैसे वर्तमान diff) शामिल कर सकती हैं ताकि Claude लाइव स्टेट पर काम करे। |
| **फ़ाइलों को रेफरेंस करें** | फ़ाइलों की ओर इशारा करें ताकि उनका कंटेंट शामिल हो जाए। |
| **फ्रंटमैटर** | `description` (मेन्यू में दिखाया जाता है) और अन्य विकल्प। |

आर्ग्युमेंट्स, bash और फ़ाइल रेफरेंस के सटीक सिंटैक्स के लिए [आधिकारिक डॉक्स](https://code.claude.com/docs/en/slash-commands) देखें।

## चुराने लायक कमांड्स

एक स्टार्टर सेट हमारी [स्लैश कमांड लाइब्रेरी](/docs/templates/slash-commands) में मौजूद है: commit message, PR description, code review, changelog, scaffold-a-component।

## कमांड बनाम स्किल बनाम सबएजेंट

काम को पैकेज करने के तीन तरीके — अंतर इसमें है कि *इसे कौन ट्रिगर करता है* और *यह कितने अलग-थलग रूप में चलता है*:

- **स्लैश कमांड** — *आप* किसी ज्ञात वर्कफ़्लो को माँग पर ट्रिगर करते हैं।
- **[स्किल](/docs/claude-code/skills)** — Claude प्रासंगिक होने पर *अपने आप* विशेषज्ञता लोड कर लेता है।
- **[सबएजेंट](/docs/claude-code/subagents)** — काम का एक हिस्सा किसी अलग-थलग एजेंट को सौंप देता है।
