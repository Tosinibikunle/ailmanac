वह ट्रैक चुनें जो सबसे ज़्यादा आपके जैसा लगता है। हर एक एक क्रमबद्ध रास्ता है — चरणों को क्रम में करें, जो आप पहले से जानते हैं उसे छोड़ दें। समय मोटे अनुमान हैं।

<Callout type="objectives" items={["वह एकमात्र ट्रैक खोजें जो आपके लक्ष्यों से मेल खाता है — नवागंतुक, ज्ञान कर्मी, बिल्डर, या टीम लीड", "कहाँ से शुरू करें इसका अंदाज़ा लगाने के बजाय एक स्पष्ट, क्रमबद्ध पढ़ने के रास्ते पर चलें", "जानें कि हर ट्रैक में मोटे तौर पर कितना समय लगता है (~45 मिनट से ~आधे दिन तक)", "अगर कोई भी ट्रैक स्पष्ट रूप से फिट नहीं लगता तो कहाँ से शुरू करें"]} />

:::tip पता नहीं कौन-सा ट्रैक आपका है?
अपने **लक्ष्य** के आधार पर मेल मिलाएँ, अपने नौकरी के पद के आधार पर नहीं। बिना कोडिंग के नतीजे चाहिए? नवागंतुक। असली काम तेज़ी से करना है? ज्ञान कर्मी। Claude Code या API को अनुकूलित करना है? बिल्डर। दूसरों तक AI पहुँचाना है? टीम लीड।
:::

:::
