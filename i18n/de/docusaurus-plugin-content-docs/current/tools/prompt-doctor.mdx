Füge einen Prompt ein, den du bereits geschrieben hast. Der Prompt Doctor bewertet ihn anhand der Prompting-Praktiken, die für Claude wichtig sind, markiert genau, was schwach ist, verlinkt jeden Befund mit der Seite, die das Problem behebt, und liefert dir ein verbessertes Grundgerüst zum Kopieren.

Er läuft **zu 100 % in deinem Browser** — dein Prompt wird niemals irgendwohin gesendet. Die Prüfungen sind einfache Heuristiken, kein KI-Aufruf: schnell, kostenlos und transparent.

<PromptDoctor />

## Was er prüft

Jede Regel verweist auf eine vertiefende Seite auf dieser Website:

| Prüfung | Warum es wichtig ist |
| --- | --- |
| **Genug, um handeln zu können** | Einzeilige Prompts zwingen Claude dazu, deine Absicht zu erraten. |
| **Legt eine Rolle fest** | "Du bist ein erfahrener X" fokussiert Ton und Urteilsvermögen deutlich — siehe [Rollen](/docs/foundations/roles). |
| **Nennt eine klare Aufgabe** | Ein vorangestelltes Imperativ-Verb ("Fasse zusammen…", "Extrahiere…") beseitigt Mehrdeutigkeit. |
| **Gibt die Ausgabeform an** | Ohne ein Format erhältst du beliebige Länge und Struktur, die Claude wählt — siehe [Strukturierte Ausgabe](/docs/api/structured-output). |
| **Zeigt ein Beispiel** | Bei Musteraufgaben schlägt ein Paar aus Eingabe→Ausgabe ganze Absätze an Beschreibung — siehe [Few-Shot](/docs/prompting/few-shot). |
| **Strukturiert, wenn lang** | Lange Prompts brauchen [XML-Tags](/docs/prompting/xml-tags), damit Claude Anweisungen von Inhalten unterscheiden kann. |
| **Schützt vor erfundenen Antworten** | Wenn du Quelltext bereitstellst, weise Claude an, nur diesen zu verwenden — die zentrale Abwehr gegen [Halluzinationen](/docs/foundations/hallucinations). |
| **Vermeidet vage Qualifizierer** | "gut", "ansprechend", "professionell" sind nicht messbar. Ersetze sie durch überprüfbare Kriterien. |

:::tip Von der Bewertung zum Einsatz
Eine hohe Bewertung ist nicht das Ziel — ein verlässliches Ergebnis schon. Behandle das Grundgerüst als Ausgangspunkt und belege es dann mit einer echten [Eval](/docs/foundations/evals), bevor du dich auf die Ausgabe verlässt.
:::
