Выберите трек, который больше всего похож на вас. Каждый из них — упорядоченный путь: проходите шаги по порядку, пропускайте то, что уже знаете. Время указано приблизительно.

<Callout type="objectives" items={["Найдите единственный трек, который соответствует вашим целям — новичок, работник умственного труда, строитель или руководитель команды", "Следуйте чёткому, упорядоченному пути чтения вместо угадывания, с чего начать", "Узнайте, сколько примерно времени занимает каждый трек (от ~45 мин до ~½ дня)", "С чего начать, если ни один из треков не кажется очевидным выбором"]} />

:::tip Не уверены, какой трек ваш?
Ориентируйтесь на свою **цель**, а не на должность. Хотите результатов без программирования? Новичок. Быстрее делать реальную работу? Работник умственного труда. Настраиваете Claude Code или API? Строитель. Внедряете ИИ для других? Руководитель команды.
:::

## Треки с одного взгляда

| Трек | Вы… | Время | Первый шаг |
| --- | --- | --- | --- |
| 🌱 Любопытный новичок | Хорошо используете ИИ, без кода | ~45 мин | [Ваши первые 5 минут](/docs/start-here/your-first-5-minutes) |
| 💼 Работник умственного труда | Ускоряете реальную работу | ~2 ч | [Библиотека шаблонов промптов](/docs/prompting) |
| 🛠️ Строитель / разработчик | Настраиваете Claude Code + API | ~½ дня | [Что такое Claude Code](/docs/claude-code/what-is-claude-code) |
| 👥 Руководитель команды | Внедряете ИИ в команде | ~3 ч | [Плагины и маркетплейсы](/docs/claude-code) |

:::note Треки опираются друг на друга
Трек «Работник умственного труда» начинается со всего трека «Новичок». Трек «Руководитель команды» начинается с основ «Строителя» (шаги 1–3). Если вы прошли один трек, следующий по уровню в основном уже сделан.
:::
