## Как они работают

Команда — это просто Markdown-файл. Положите его в каталог команд, и его имя файла станет командой. Тело файла — это промпт, который Claude выполняет, когда вы её вызываете.

*Расположение* файла определяет его область действия:

<Callout type="tip" items={["Именем команды становится имя файла, а не frontmatter. changelog.md — это всегда /changelog."]} />

## Минимальный пример

Вот полный файл команды. Frontmatter содержит `description`; всё, что ниже, — это промпт.

<PromptCard title="commit.md — составить Conventional Commit">{`---
description: Draft a Conventional Commit message from the staged diff
---

Look at the staged changes with \`git diff --cached\` and write a single
Conventional Commit message (feat/fix/docs/refactor/chore). Output only the
message, no preamble.`}</PromptCard>

Сохраните это как `commit.md`, и `/commit` будет делать это каждый раз. (Назовите команду так, как назван файл — то же тело, сохранённое как `changelog.md`, станет `/changelog`.)

## Полезные строительные блоки

Четыре ингредиента превращают статический промпт в гибкую команду:

| Строительный блок | Что он делает |
| --- | --- |
| **`$ARGUMENTS`** | Подставляет то, что пользователь набрал после команды. `/review src/auth.ts` → путь попадает в ваш промпт. |
| **Запуск bash и встраивание вывода** | Команды могут включать вывод оболочки (например, текущий diff), чтобы Claude действовал по актуальному состоянию. |
| **Ссылки на файлы** | Указывайте на файлы, чтобы включить их содержимое. |
| **Frontmatter** | `description` (показывается в меню) и другие опции. |

Точный синтаксис аргументов, bash и ссылок на файлы смотрите в [официальной документации](https://code.claude.com/docs/en/slash-commands).

## Команды, которые стоит позаимствовать

Стартовый набор есть в нашей [Библиотеке слэш-команд](/docs/templates/slash-commands): сообщение коммита, описание PR, код-ревью, changelog, заготовка компонента.

## Команда против навыка против субагента

Три способа упаковать работу — разница в том, *кто это запускает* и *насколько изолированно это выполняется*:

- **Слэш-команда** — *вы* запускаете известный рабочий процесс по требованию.
- **[Навык](/docs/claude-code/skills)** — Claude загружает экспертизу *автоматически*, когда это уместно.
- **[Субагент](/docs/claude-code/subagents)** — делегируйте часть работы изолированному агенту.
