挑一个最像你的赛道。每条都是有序的路径——按顺序完成各个步骤，已经会的就跳过。时间只是大致估算。

<Callout type="objectives" items={["找到唯一符合你目标的赛道——新手、知识工作者、构建者，或团队负责人", "沿着清晰、有序的阅读路径前进，而不是凭猜测从哪里开始", "大致了解每条赛道需要多少时间（约 45 分钟到约半天）", "如果没有一条赛道明显适合你，该从哪里入手"]} />

:::tip 不确定哪条赛道是你的？
按你的**目标**来匹配，而不是按职位头衔。想要不写代码就拿到结果？好奇的新手。想让真正的工作更快？知识工作者。想定制 Claude Code 或 API？构建者。想把 AI 推广给其他人？团队负责人。
:::

## 赛道一览

| 赛道 | 你是…… | 时间 | 第一步 |
| --- | --- | --- | --- |
| 🌱 好奇的新手 | 把 AI 用好，无需写代码 | 约 45 分钟 | [你的最初 5 分钟](/docs/start-here/your-first-5-minutes) |
| 💼 知识工作者 | 加速真正的工作 | 约 2 小时 | [提示词模式库](/docs/prompting) |
| 🛠️ 构建者 / 开发者 | 定制 Claude Code + API | 约半天 | [Claude Code 是什么](/docs/claude-code/what-is-claude-code) |
| 👥 团队负责人 | 向团队推广 AI | 约 3 小时 | [插件 & 市场](/docs/claude-code) |

:::note 赛道是层层递进的
知识工作者赛道以整条新手赛道作为起点。团队负责人赛道以构建者的核心（步骤 1–3）作为起点。如果你完成了一条赛道，那么上一层的赛道大部分其实已经做完了。
:::
