Pick the track that sounds most like you. Each is an ordered path — do them in order, skip what you already know. Times are rough.

## 🌱 The Curious Newcomer · ~45 min

You want to use AI well without becoming a programmer.

1. [Your First 5 Minutes](/docs/start-here/your-first-5-minutes) <LevelBadge level="beginner" />
2. [Which Claude Should I Use?](/docs/start-here/which-claude) <LevelBadge level="beginner" />
3. [Prompting Basics](/docs/prompting/basics) <LevelBadge level="beginner" />
4. [What Is an LLM?](/docs/foundations) <LevelBadge level="beginner" />
5. [Everyday Productivity](/docs/playbooks/productivity) <LevelBadge level="beginner" />

## 💼 The Knowledge Worker · ~2 hrs

You want AI to make your real job faster — writing, research, analysis.

1. The Curious Newcomer track (above), then…
2. [Prompt Patterns Library](/docs/prompting) <LevelBadge level="all" />
3. [Hallucinations & How to Reduce Them](/docs/foundations) <LevelBadge level="intermediate" />
4. Your playbook: [Writing](/docs/playbooks), [Research](/docs/playbooks), or [Data Analysis](/docs/playbooks)
5. [Projects: persistent workspaces](/docs/claude-app) and [Reusable Prompt Templates](/docs/templates)

## 🛠️ The Builder / Developer · ~½ day

You write code and want to customise Claude Code and build on the API.

1. [What Claude Code Is](/docs/claude-code/what-is-claude-code) <LevelBadge level="beginner" />
2. [CLAUDE.md & Memory Files](/docs/claude-code) <LevelBadge level="beginner" /> + [Plan Mode](/docs/claude-code)
3. [Permissions](/docs/claude-code) and [settings.json](/docs/claude-code) <LevelBadge level="intermediate" />
4. [Your First API Call](/docs/api/first-call) → [Tool Use](/docs/api) → [Building Agents](/docs/api) <LevelBadge level="advanced" />
5. [Walkthrough: Customise Claude Code for a real repo](/docs/walkthroughs) <LevelBadge level="intermediate" />
6. [Security & Responsible Use](/docs/security) <LevelBadge level="advanced" />

## 👥 The Team Lead · ~3 hrs

You're rolling AI out to a team and care about consistency, safety, and cost.

1. The Builder track essentials (1–3 above)
2. [Plugins & Marketplaces](/docs/claude-code) — ship a shared toolkit <LevelBadge level="advanced" />
3. [Cost & Latency Tradeoffs](/docs/foundations) and [Choosing a Model](/docs/api/choosing-a-model) <LevelBadge level="intermediate" />
4. [Securing Agents](/docs/security) and [Hardening Autonomous Runs](/docs/security) <LevelBadge level="advanced" />
5. [Responsible Use & Verification](/docs/security) — set team norms <LevelBadge level="all" />

---

Not sure which fits? Start with [Your First 5 Minutes](/docs/start-here/your-first-5-minutes) — it's useful no matter where you end up. And read [How Levels & Freshness Work](/docs/start-here/levels-and-freshness) so the badges and dates make sense.
