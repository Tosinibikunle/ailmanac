# Changelog

All notable changes to AILmanac are documented here. The format is based on
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/).

This tracks changes to the **site itself** (structure, tooling, major content
drops). For a plain-English digest of what changed in *Claude* and the *API*,
see the [What's New](docs/whats-new/) section of the site.

## [Unreleased]

### Added
- Initial Docusaurus scaffold: brand, homepage level-picker, local search, Mermaid.
- Custom MDX components: `LevelBadge`, `VerifyNote`, `ModelTable` (registered globally).
- Full information architecture (12 sections, 100+ pages) with autogenerated sidebar.
- `data/models.json` as the single source of truth for current Claude models,
  rendered with a "last verified" stamp.
- Extended-MVP content across Start Here, Foundations, Prompting, Claude.ai,
  Claude Code, API, Playbooks, Templates, Security, and Contribute.
- Governance: dual license (CC BY 4.0 + MIT), trademark NOTICE, Code of Conduct,
  Contributing guide, issue/PR templates, CODEOWNERS, Dependabot.
- CI: production build check (broken links fail), deploy to GitHub Pages,
  scheduled link checking.
